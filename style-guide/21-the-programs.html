02 · The idea

One clean directory of everything DiscNW runs

DiscNW is big. It has served more than 10,000 players a year since it started at Potlatch in 1995. It runs leagues, camps, and tournaments, for kids and for adults, in every season.

That is a lot to hold on one website. Today you have to already know the name of the thing you want before you can find it. A new parent and a twenty-year captain get the same long list of links.

The fix is a directory. Think of a channel guide or an app grid: every program on its own tile, each with its real logo, its season, and who it is for. You choose a season and who is playing, and the wall narrows to the programs that fit. A new family or a first-time adult player should find the right one in a few seconds.

Why this fits

  • DiscNW already has strong, distinct program logos. This design lets them do the work.
  • The hard part of a big org is finding the right door. A directory is the answer to that.
  • Staff can add a program by adding a tile. No new page design each time.
  • The palette stays quiet on purpose, so the logos carry the color.

Plain goal: a parent types nothing and reads nothing long. They pick "Summer" and "Youth" and see four options with logos they may already recognize.

03 · Palette

Quiet on purpose, so the logos carry the color

The page is built from neutrals. Two grey grounds, white tiles, and one calm slate-blue accent for the parts you click. Every program logo is already colorful, so the interface stays out of their way. The accent does one job: it marks what is interactive.

  • Graphite Ink

    #1B1E23

    Dark sections, the header, and headline text.

  • Paper Grey

    #ECEBE8

    The page ground. Warm, not clinical, so white tiles pop.

  • Tile White

    #FFFFFF

    The logo tiles. One neutral ground for every program.

  • Slate Blue

    #35607F

    The one accent. Buttons, links, and active filters.

  • Slate Deep

    #284B63

    The accent when pressed or hovered. Same family, darker.

  • Reading Grey

    #565B62

    Secondary text, seasons, and who-it-is-for lines.

04 · Typography

Clean and legible, directory-grade

A directory has to be easy to scan, so the type is plain and clear. Hanken Grotesk sets the names and headings. Inter sets the body. JetBrains Mono handles the small parts a guide needs: channel numbers, seasons, and short labels. All three load from Google Fonts.

Names & headings

Choose this direction if…

You want the size of DiscNW to feel like a strength, not a maze. A directory puts every program one glance away, lets the real logos carry the color, and gives staff a place to add the next program without redesigning a page.

Think twice if…

You want the homepage to lead with one big story or one photograph. This direction leads with a grid. It is built to help people find and sort, so it reads more like a useful tool than a single bold statement.
